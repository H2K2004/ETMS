# ETMS - Employee Task Management System

A web-based system built with PHP for managing employee tasks and leave requests.

## Features
- Approve/Reject Leave Requests
- Create, Edit, Delete Tasks
- Manage Tasks
- View Leave Records

## Technologies Used
- PHP
- Bootstrap (CSS)

## How to Run
1. Clone the repository
2. Import the database
3. Run on a local server (XAMPP/WAMP)
